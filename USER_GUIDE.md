# InboxBar User Guide

**InboxBar** is a sleek, docked sidebar for Microsoft Outlook that keeps your emails, tasks, and schedule visible without cluttering your workspace. Designed for productivity, it sits quietly on the edge of your screen, notifying you of what matters most.

---

## 1. Getting Started

### Installation
1.  Locate the verified `InboxBar.exe` file.
2.  Double-click to launch. The app will dock to the right side of your primary monitor.
3.  **Note:** InboxBar requires the **classic Microsoft Outlook desktop application** to be running in the background.

![InboxBar Main View](placeholder_main_view.png)

### The "Hot Strip" & Auto-Hide
InboxBar is designed to be unobtrusive.
-   **Minimize:** Click the **Pin icon** in the top bar to hide the sidebar. It shrinks to a thin "Hot Strip" on the screen edge.
-   **Restore:** Hover your mouse over the colored edge strip to bring the full sidebar back.
-   **Pin:** Click the **Pin icon** in the top-right to keep the sidebar always visible.

---

## 2. Interface Overview

### Top Bar
Located at the very top of the sidebar, this area contains global controls:
-   **Spanner Icon:** Opens the **Settings Panel**.
-   **Refresh Icon:** Manually checks for new emails immediately.
-   **Pin Icon:** Toggles between "Auto-Hide" and "Always Visible" modes.
-   **Help Icon (?):** Opens this User Guide.

![Top Bar Controls](placeholder_top_bar.png)

### Bottom Bar
Located at the bottom of the sidebar:
-   **Red Cross (Left):** Completely closes the InboxBar application.
-   **Plus Button (+):** Opens the **Quick Actions** menu (New Email, New Meeting, etc.) or executes immediately if only one option is configured.
-   **Calendar Button:** Opens the calendar view of outlook
-   **Envelope Button:** Opens the email view of outlook

![Bottom Bar Controls](placeholder_bottom_bar.png)

### Visual Notifications (The "Pulse")
The sidebar edge glows ("pulses") to notify you of changes without sounds.
-   <span style="color:#007ACC">**Blue Pulse:**</span> New Unread Email.
-   <span style="color:#FFAA00">**Orange Pulse:**</span> Upcoming Meeting (starts in <15 mins) or Reminder.
-   <span style="color:#28A745">**Green Pulse:**</span> Active Task notification.

## 3. Customizing Settings
Click the **Spanner Icon** in the Top Bar to open the Settings Panel.

### General Settings
-   **Font Family & Size:** Customize the text appearance.
-   **Refresh Rate:** How often the app checks Outlook (default is 30s).

### Email List Options
Customize how emails appear in the list:
-   **Include Read Email:** **Unchecked by default.** Check this if you want to see *all* emails. Leave unchecked to see only Unread items (Inbox Zero style).
-   **Show if has Attachment:** Displays a paperclip icon if files are attached.
-   **Email Window Content:** Choose to show Sender, Subject, and Content Body lines.
-   **Show Content on Hover:** Hover over an item to see a popup preview of the full message.

![Email Settings](placeholder_email_settings.png)

### Interactions
-   **Show Buttons on Hover:** Reveals action buttons (Delete, Mark Read) only when you mouse over an email.
-   **Double Click to Open:** Enables opening the email in the full Outlook window by double-clicking.

### Button Configuration & Folder Selection
You can customize the **4 Action Buttons** displayed on each email card.
1.  **Select Action:** Choose an action like *Move To...*, *Mark Read*, or *Reply*.
2.  **Folder Selection:**
    -   If you select the **Move To...** action, you can select a folder alongside.
    -   Click the `...` button to open the **Folder Picker**.
    -   Select the target Outlook folder where emails should be moved when this button is clicked.

![Button Configuration](placeholder_button_config.png)

### Reminder Settings (Dual Mode)
If you are using the "Emails & Reminders" Split View:
-   **Follow-up Flags:** Show flagged emails due Today, Tomorrow, etc.
-   **Categories:** Filter items by Outlook Category colors.
-   **Importance:** Show only High Importance items if desired.
-   **Meetings:** Toggle Pending vs. Accepted meetings.
